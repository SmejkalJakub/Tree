Tree.Tree(evalFunction)

new Tree(evalFunction)

Parameters:
Name Type Description
evalFunction function This functions decides if the new value should be added to the left or right subtree. It should have 2 parameters and return True/False. If the function returns True, new value will be added to the left subtree, otherwise it will be added to the right subtree.

insertNode(node, newNode)

Inserts new node to the tree. If the node should be inserted to the left or right subtree is decided on evalFunction. If the selected child is empty, new node is inserted there. Otherwise this function is recursively called on selected subtree.
Parameters:
Name Type Description
node Node parent node on which the new node should be attached to
newNode Node new node that should be inserted

insertValue(value)

This function takes care of inserting new nodes to the tree. If there is no root it will just create it. Otherwise it will start recursion that will insert the node to the tree based on the specified evalFunction.
